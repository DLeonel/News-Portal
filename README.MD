# 📰 Portal de Notícias

Um blog/site de notícias desenvolvido para praticar conceitos fundamentais de desenvolvimento web, integração com banco de dados e operações CRUD.

## 🎯 Objetivo

Este projeto foi criado com foco em aprendizagem e aplicação prática de conceitos Full Stack, permitindo exercitar conhecimentos de:

* Estruturação de interfaces web;
* Manipulação do DOM;
* Comunicação entre Front-end e Back-end;
* Integração com banco de dados;
* Autenticação básica;
* Operações CRUD (Create, Read, Update e Delete).

## 🚀 Funcionalidades

### Área Pública

Os visitantes podem:

* Visualizar todos os artigos publicados;
* Acessar a leitura completa de qualquer artigo;
* Receber sugestões de novos conteúdos ao final de cada leitura;
* Registrar ou confirmar um endereço de e-mail.

### Área Administrativa

O administrador possui acesso a um painel de gerenciamento que permite:

* Criar novos artigos;
* Editar artigos existentes;
* Excluir artigos;
* Alterar o nome do site;
* Alterar o ícone (favicon);
* Modificar o texto do rodapé.

## 🔮 Funcionalidades Futuras

Algumas funcionalidades poderão ser adicionadas em versões posteriores:

* Sistema de comentários;
* Sistema de curtidas (likes);
* Categorias de artigos;
* Pesquisa de conteúdo;
* Perfil de utilizador.

## 🛠️ Tecnologias Utilizadas

### Front-end

* HTML5
* CSS3
* JavaScript
* Tailwind CSS
* Bootstrap Icons

### Back-end

* Node.js
* Express.js
* MySQL2
* CORS

### Banco de Dados

* MySQL

## 📂 Estrutura da Aplicação

### Página Inicial

Apresenta a lista de artigos disponíveis, permitindo ao utilizador navegar rapidamente pelos conteúdos publicados.

### Página de Leitura

Exibe o artigo completo juntamente com sugestões de outros conteúdos relacionados.

### Painel Administrativo

Centraliza todas as operações de gestão do site, incluindo administração de artigos e personalização das informações institucionais da plataforma.

## 📚 Finalidade

Este projeto foi desenvolvido para consolidar conhecimentos em desenvolvimento web moderno, explorando a integração entre interface, servidor e banco de dados através de uma aplicação funcional e organizada.
